1. O QUE É O NEXUSAGENT AI
===============================================================

O NexusAgent AI é um agente autônomo de programação que integra
15 provedores de IA gratuitos do mundo. Ele permite:

- Executar tarefas de programação com IA (geração, revisão, etc.)
- Rodar código em sandbox seguro (Python, JavaScript, Bash)
- Acessar terminal remoto com restrições de segurança
- Automatizar navegação na web
- Criar e gerenciar projetos de software
- Executar workflows automatizados
- Gerenciar system prompts personalizados
- Fazer deploy em plataformas cloud
- Consultar banco de dados diretamente
- Exportar resultados em PDF, DOCX e TXT
- Monitorar tudo em tempo real

3. COMO INSTALAR E RODAR
===============================================================

REQUISITOS:
- Python 3.10+
- pip (gerenciador de pacotes)
- Internet (para APIs de IA)

INSTALAÇÃO:
1. Extraia o arquivo ZIP
2. Instale as dependências:
   pip install fastapi uvicorn python-jose[cryptography] aiohttp python-multipart reportlab python-docx

3. Configure as API keys (opcional — Groq e OpenRouter funcionam sem setup):
   export GROQ_API_KEY="sua-chave-aqui"
   export OPENROUTER_API_KEY="sua-chave-aqui"
   export GOOGLE_API_KEY="sua-chave-aqui"
   export HF_TOKEN="sua-chave-aqui"

4. Inicie o servidor:
   cd backend
   python3 -m uvicorn main:app --host 0.0.0.0 --port 8000

5. Acesse no navegador:
   http://localhost:8000

PARA ACESSO EXTERNO (Cloudflare Tunnel):
   cloudflared tunnel --url http://localhost:8000

LOGIN PADRÃO:
   Usuário: admin
   Senha: admin123

4. COMO USAR (GUIA COMPLETO)
===============================================================

DASHBOARD:
- Visão geral com estatísticas
- Ações rápidas para tarefas comuns
- Tarefas recentes

NOVA TAREFA:
1. Selecione o tipo de tarefa (18 opções)
2. Descreva o que precisa
3. Opcionalmente selecione um system prompt
4. Arraste ou selecione arquivos (opcional)
5. Clique em "Executar"
6. A resposta aparece em streaming (letra por letra)
7. Copie, baixe em PDF/DOCX/TXT

HISTÓRICO:
- Lista todas as tarefas executadas
- Clique para ver detalhes e versões
- Exporte tudo como TXT

TERMINAL:
- Execute comandos bash
- Navegue pelos arquivos
- Edite arquivos diretamente
- Histórico com setas (↑↓)

SANDBOX:
- Execute Python, JavaScript ou Bash
- Veja output e erros em tempo real
- Timeout de 30 segundos

BROWSER:
- Navegue em qualquer URL
- Extraia conteúdo de páginas
- Busque no DuckDuckGo

PROJETOS:
- Crie projetos a partir de templates
- Templates: React, Next.js, Vue, Svelte, Flask, FastAPI, etc.

WORKFLOWS:
- Crie automações com múltiplos passos
- Cada passo é uma tarefa IA
- Execute e veja resultados passo a passo

SYSTEM PROMPTS:
- Crie prompts personalizados
- Associe a tipos de tarefa específicos
- Use ao executar tarefas

DEPLOYS:
- Conecte projetos a plataformas de deploy
- Suporta: Vercel, Netlify, Render, Cloudflare Pages

BANCO DE DADOS:
- Visualize todas as tabelas (14 tabelas)
- Execute queries SELECT (somente leitura)
- Navegue pelos dados

MONITORAMENTO:
- Estatísticas do sistema
- Eventos de startup/shutdown
- Saúde dos provedores

NOTIFICAÇÕES:
- Receba alertas sobre tarefas
- Marque como lidas

CONFIGURAÇÕES:
- Toggle Dark/Light mode
- Configure API keys (15 provedores)
- Gerencie sessões ativas
- Crie API keys públicas
- Altere sua senha

8. SOLUÇÃO DE PROBLEMAS
===============================================================

PROBLEMA: "Nenhuma API key configurada"
SOLUÇÃO: Vá em Configurações e adicione pelo menos uma chave
(Groq e OpenRouter são gratuitos)

PROBLEMA: Servidor não inicia
SOLUÇÃO: Verifique se a porta 8000 está livre:
  lsof -i :8000
  mate -9 <PID>

PROBLEMA: Token expirado
SOLUÇÃO: Faça login novamente

PROBLEMA: Streaming não funciona
SOLUÇÃO: Verifique se o provedor suporta streaming
(Groq e OpenRouter suportam)

PROBLEMA: Tunnel muda de URL
SOLUÇÃO: O Cloudflare gratuito gera URLs novas a cada reinício
Para URL fixa, use um domínio próprio

===============================================================
9. SEGURANÇA
===============================================================

- Terminal: blocklist de comandos perigosos
- Sandbox: execução isolada com timeout
- Browser: bloqueio de SSRF (IPs privados, metadata endpoints)
- DB: WAL mode para concorrência
- Rate limiting em endpoints críticos
- JWT com expiração de 24 horas
- Validação de entrada em todos os endpoints
- Headers de segurança (XSS, CSRF, nosniff)
- Backup automático do banco de dados
